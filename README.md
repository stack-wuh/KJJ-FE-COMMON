# KJJ-FE-COMMON

## 使用说明 

现在我们使用的是dumi开发工具, 它是一款开箱即用, 专注于开发组件和文档的框架。


你可以使用npm, 也可以使用yarn, 作为你的包管理器。

其使用步骤如下:

1. 检出dev分支, 现在我们走github开发流
```bash
git clone http://139.129.102.100/gitlab/shadow/kjj-fe-common.git -b dev

cd KJJ-FE-COMMON

npm i 
```

2. 可以直接进入开发模式, 浏览现在的组件文档
```bash

npm run start

```
